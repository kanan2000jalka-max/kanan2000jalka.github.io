# kanan2000jalka.github.io
Тестовая текстовая игра написанная нейросетью

Краткое объяснение структуры:
  
  HTML (index.html) - Каркас
    
  1. Базовый HTML-документ
  
  2. Подключение Telegram Web App SDK
  
  3. Контейнеры для фона, текста и выбора
  
  CSS (style.css) - Внешний вид
  
  1. Адаптивный дизайн под Telegram
  
  2. Анимации для плавности
  
  3. Градиенты и тени для атмосферы
  
  JavaScript (game.js) - Логика
  
  1. Блок 1: Инициализация Telegram Web App
  
  2. Блок 2: База данных всех сцен (текст, фон, выборы)
  
  3. Блок 3: Основные функции игры (загрузка сцен, обновление статистики)
  
  4. Блок 4: Специальные сцены (Game Over)
15. Блок 5: Запуск игры при загрузке
16. Блок 6: Система сохранения (опционально)
