# kanan2000jalka.github.io
Тестовая текстовая игра написанная нейросетью

Краткое объяснение структуры:
1. HTML (index.html) - Каркас
2.     Базовый HTML-документ
    Подключение Telegram Web App SDK
    Контейнеры для фона, текста и выбора
3. CSS (style.css) - Внешний вид
    Адаптивный дизайн под Telegram
    Анимации для плавности
    Градиенты и тени для атмосферы
4. JavaScript (game.js) - Логика
    Блок 1: Инициализация Telegram Web App
    Блок 2: База данных всех сцен (текст, фон, выборы)
    Блок 3: Основные функции игры (загрузка сцен, обновление статистики)
    Блок 4: Специальные сцены (Game Over)
    Блок 5: Запуск игры при загрузке
    Блок 6: Система сохранения (опционально)
